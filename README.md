# skills

A collection of my own personal Claude skills. 

## Claude Code 

These should go in `~/.claude/skills`.

Installing these on a new computer:

```sh
cd ~/.claude/
git clone git@github.com:stephenturner/skills.git
```

## Claude.ai / Claude Desktop

First, zip a particular directory you want. E.g.:

```sh
zip -r deslop.skill deslop
```

Then add to Claude.
